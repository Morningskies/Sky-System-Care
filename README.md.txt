# 🛡️ Sky System Care

**Sky System Care** is a lightweight desktop utility designed for everyday PC users, offering 1-click optimization, memory cleanup, and smart system monitoring.

## ✨ Features
- 🚀 One-click system optimization
- 🧠 Real-time RAM & CPU monitoring
- 🔄 Auto-update checker
- 🌙 Dark mode UI
- 🧹 Deep system cleanup (temp files, cache, etc.)

Sky System Care includes a built-in update checker.

- It reads the current version from:
  [`version.txt`](https://raw.githubusercontent.com/Morningskies/Sky-System-Care/main/version.txt)

- If a new version is found, users are redirected to download:
  [SkySec_Installer.exe](https://github.com/Morningskies/Sky-System-Care/releases/latest/download/SkySec_Installer.exe)
